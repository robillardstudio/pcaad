Remapping ranges of data again:

  • bigger number? more blue, bigger circle, taller triangle
  • mapping data to ranges appropriate for color, speed, size, etc
  • map
    • percentage data remapped to rotation: “map(value, 0,100, 0,360)”
    • temperature data to color: “map(value, 32,99, 50,255)”
    • easier than writing: (((99-32) / 67) * 205) + 50
    • map(n, -1.0, 1.0, 0,255);
  • map example - editor
