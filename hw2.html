Assignment 2: functions, parameters, and iteration

Due Tuesday February 4th before class.

While the first assignment asked you to draw one thing, the second assignment challenges you to create a collection of similar things. You will use functions, arguments, and iteration, to generate a visual collection of objects/items/sketches/etc, controlled by parameters. E.g., a collection of buildings, faces, animals, etc. Each is built from a set of components but sizes/parts/colors/relationships are each different.

You will need to:

  • create “functions” to abstract repeatable sets of drawing commands
  • create “arguments” to pass parameters into the functions to make individual elements different from one another when drawn
  • use “for” or “while” loops to repeat visual drawn elements
  • change the arguments passed to functions so what is drawn changes
  • document the diversity of possibilities using different argument values
  • Consider multiple drawing layers
  • Consider using random() or noise() functions

While not required, if you’re comfortable with all of the above, a nice stretch goal would be to create both “high” and “low” level parameters:

  • low-level parameters control individual single visual traits
  • high-level parameters changes multiple visual traits
  • e.g., “leftEyeWidth” vs “personAge”
