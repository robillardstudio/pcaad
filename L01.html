Processing

We’ll be using a programming language called “Processing”. It has a long history as a tool that allows artists and designers to “sketch” with code.

Processing is usually implemented using another programming language, making it much easier to do things like draw shapes and pictures than it would be in that language otherwise.

We’re going to use “p5.js” which is Processing implemented in Javascript, the main programming language used inside web browsers.

Brief History of Processing, links, books, resources:

  • DBN, Processing, processing.js, processing for android, processing.py, p5.js
  • Processing.js vs P5.js
  • Make: Getting Started with p5.js by Lauren McCarthy, Casey Reas, and Ben Fry
  • Showcase
  • Community Sketches
