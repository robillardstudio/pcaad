Programming Libraries

Overview

(Reminder that the new topics I’m introducing are NOT required for use in your final two assignments. Just things I want you to know about.)

Instead of writing all of the code you need by yourself, programmers frequently turn to “libraries” of code created by other people.

These are collections of functions usually centered around some theme like sound or making games. You can think of “p5.js” as a javascript library that provides a set of functions to help artists and designers learn how to program and make interactive visual web projects.

Programmers have contributed a large number of libraries to work with p5.js.

Read Using a library at the top of the linked page.

Browse through the list of p5.js libraries at the bottom of the linked page.

To USE a library you either need to download the files and put them in your folder, or find someplace where they live on the web to point at with a URL.

A library is often one file: in our case a Javascript file.

Exercise:

  • choose and use a library!
  • I’d recommend you try a simpler one first like “scribble” or “play”

  • try to modify their example slightly
  • try to make something SUPER simple yourself using functions provided by the library
