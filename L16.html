General Concepts

Here are some terms we’ll be using followed by some examples:

  • “objects” : “Circle”, “Person”, “Building”, “Face”
  • “properties” : circle2.radius, person.name = “frank”, face3.smiling = true
  • “constructors” : creating an instance
    • var p = new Person(“bob”);
  • instances: different individuals with different property values
  • “methods” : circle.position(x,y), shape.setColor(c), face.draw()
  • inheritance:
    • a Shape has a position and a color
    • a Circle is a Shape (it inherits position and color) but it also adds a radius
    • a Building is a Shape and it also adds numStories and numWindows

We won’t be covering “inheritance” in this class, but you might see it in the wild.
