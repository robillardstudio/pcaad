Assignment 1: draw with code

Due Thursday Jan 16th before class (hand in via Carmen.)

This first assignment is intended for you to quickly gain experience with the iterative process of writing, testing, and modify code. You’ll gain experience learning new tools, making appropriate use of example code and documentation, and get a first taste of “parametric design”.

You are to only use the tools and techniques you are introduced to in the first few classes to create drawings using shapes, lines, and colors.

For this assignment you must use each of these:

  • variables you create yourself (not just built-in variables)
  • a couple built-in math functions
  • conditional statements (if/then/else).

Please do NOT use techniques we haven’t covered in class yet (e.g. no while/for loops, no custom functions, arrays, object methods or classes, etc.)

Demonstrate breadth of experimentation (try a bunch of things!) instead of crafting One Perfect Masterpiece from one or two things. Document your effort (show your work, not just your final product.)

For what you should submit, what to post to Carmen, how I will grade, and other advice, please read Submission Instructions and Grading.
